## Тест-кейсы

### TC1. Создание задачи

**ID:** TC1

**Предусловия:**
1. Открыть браузер Google Chrome
2. Открыть главную страницу сайта по адресу https://avito-tech-internship-psi.vercel.app

**Шаги проверки:**
1. Нажать на кнопку "Создать задачу"
2. Заполнить поле "Название"
3. Заполнить поле "Описание"
4. Нажать на выпадающий список "Проект" и нажать на один из проектов
5. Нажать на выпадающий список "Приоритет" и нажать на один из приоритетов
6. Нажать на выпадающий список "Исполнитель" и нажать на одного из исполнителей
7. Нажать на кнопку "Создать"

**Ожидаемый результат:** Задача с введенными и выбранными данными появилась в списке задач.

### TC7. Переход на доску проекта

**ID:** TC7

**Предусловия:**
1. Открыть браузер Google Chrome
2. Открыть главную страницу сайта по адресу https://avito-tech-internship-psi.vercel.app

**Ожидаемый результат:** Открывается страница по адресу https://avito-tech-internship-psi.vercel.app/board/{id}, где id - номер доски, название доски соответствует выбранному проекту
