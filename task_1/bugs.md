# Список багов на странице карьерного сайта Авито

## High priority

**Почему high:** критически нарушают основную функциональность поиска и фильтрации вакансий — пользователи видят неверные результаты или противоречивую информацию, что делает сайт практически бесполезным для поиска работы.

1. Фильтр "Направление" не исключает вакансии других направлений из результатов поиска при выборе "Data Science"
2. Фильтр "Город" не исключает вакансии из других городов из результатов поиска при выборе "Санкт-Петербург"
3. Фильтр "Формат работы" не исключает вакансии с другими форматами работы из результатов поиска при выборе "Можно удаленно"
4. Фильтр "Руководящая роль" не исключает вакансии с другой ролью из результатов поиска
5. Блок "Ничего не нашлось" отображается одновременно с карточками вакансий

---

## Medium priority

**Почему medium:** вводят в заблуждение пользователя, показывают некорректные или неполные данные о вакансиях, снижают доверие и удобство, но не полностью ломают поиск.

1. Не отображается кнопка очистки в строке поиска
2. Лишний фильтр тег "Поддержка пользователей" отображается в фильтрах при выборе направления Data Science
3. Лишний фильтр тег "Аналитик" отображается в фильтрах при выборе направления Data Science
4. Значение в счётчике вакансий рядом с направлением "Клиентский сервис" не соответствует реальному количеству вакансий
5. Несоответствующий тег команды "Авто" отображается под вакансией "Руководитель аналитики в транзакционное направление жилой недвижимости"
6. Несоответствующий тег команды "Недвижимость" отображается под вакансией "Аналитик данных в бизнес-команду Авито Авто"
7. Не отображается город напротив вакансии "Аналитик данных в бизнес-команду Авито Авто"

---

## Low priority

**Почему low:** косметические, визуальные или мелкие текстовые недочёты; не влияют на основную функциональность и поиск вакансий, но портят впечатление от сайта.

1. Подсказка для фильтра "Команда" отображает "Вакансий пока нет" вместо "Нет команд для этих вакансий"
2. Кнопки фильтров "Направление", "Город", "Команда" и "Формат работы" не выровнены вертикали
3. Отсутствует выравнивание форматов работы "офис" и "удаленно" по правому краю
4. Отсутствует иконка стрелки в кнопке "Про направление" напротив названия блока "Аналитика данных"
5. Грамматическая ошибка в футере: «227 вакансии открыто» вместо «227 вакансий открыто»
6. Орфографическая ошибка в футере: «Вакансии в Telagram» вместо «Вакансии в Telegram»
7. Неправильный адрес офиса в Москве в футере: "ул. Лесная, 77" вместо "ул. Лесная, 7"
